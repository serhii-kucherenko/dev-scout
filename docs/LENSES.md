# Research lenses

| Lens | Scouts for |
|------|------------|
| `ship-faster` | Agent loops, codegen, review automation |
| `build-robust` | Testing, eval harnesses, guardrails |
| `tooling-setups` | MCP, IDE configs, CLI workflows |
| `production-patterns` | Deploy, observability, rollback |
| `model-workflows` | Model routing, cost/latency |

Specs: `config/lenses/<id>.yaml`  
Output: `01-research/lenses/<id>/output.json`

Each lens output contains `JamItem[]` with source link and how-to steps.
